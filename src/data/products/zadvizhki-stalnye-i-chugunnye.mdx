* **30с41нж** \
  Стальная задвижка с выдвижным шпинделем для перекрытия потока в трубопроводах. Используется в системах воды, пара и газа при давлении до 1,6 МПа.
* **30с64нж** \
  Усиленная стальная задвижка с выдвижным шпинделем. Предназначена для работы в системах с высоким давлением до 4,0 МПа. Применяется в теплоэнергетике и промышленности.
* **30с964нж** \
  Задвижка 30с64нж с электроприводом. Используется для автоматизации процессов в трубопроводных системах высокого давления.
* **30с941нж** \
  Аналог 30с41нж, оснащён электроприводом. Обеспечивает дистанционное управление в системах воды, пара и газа при давлении до 1,6 МПа.
* **30ч6бр** \
  Чугунная задвижка с обрезиненным клином и невыдвижным шпинделем. Универсальное решение для систем водоснабжения и отопления (Ру 16).
* **30ч906бр** \
  Модификация 30ч6бр с электроприводом. Подходит для автоматического управления потоком в системах холодной и горячей воды.
* **30с15нж (Ру 40)** \
  Стальная задвижка для эксплуатации при давлении до 4,0 МПа. Используется на ответственных участках трубопроводов с высокой нагрузкой.
* **30с76нж (Ру 64)** \
  Усиленная задвижка стального исполнения для систем с давлением до 6,4 МПа. Применяется в энергетике и на промышленных объектах.
